# Abenezer Yirgalem — Portfolio Website

Welcome! This repository holds a static personal portfolio website for **Abenezer Yirgalem**. The site is built from the "iPortfolio" Bootstrap template and showcases personal details, skills, resume, and selected projects.

--

## Project Overview

- **Project:** Personal portfolio website
- **Purpose:** Showcase professional profile, skills, projects, and contact info
- **Author:** Abenezer Yirgalem
- **Template base:** iPortfolio (BootstrapMade)

## Features

- Responsive one-page layout with multiple sections: Home, About, Resume, Projects, Contact
- Smooth scroll, animated content, and typed text effect in the hero section
- Portfolio gallery with lightbox and project links (GLightbox + Isotope for filtering)
- Mobile-friendly, cross-browser compatible using Bootstrap 5

## Technologies & Libraries

- Core: HTML5, CSS3, JavaScript
- CSS Framework: Bootstrap 5
- Animations & UI: AOS (animate on scroll), GLightbox, Swiper
- Utilities: Isotope (layout), Typed.js (typed text effect), PureCounter
- Vendor files are included in the `assets/vendor/` folder

## Project Structure

Select files and folders:

- `index.html` — Main site page (one-page layout)
- `portfolio-details.html` — Project detail template (if used)
- `assets/css/main.css` — Custom styles
- `assets/js/main.js` — Entry script (initializes components and interactions)
- `assets/img/` — All images: avatar, backgrounds, portfolio items, icons
- `assets/vendor/` — Third-party libraries (Bootstrap, AOS, Swiper, etc.)
- `forms/` — Server-side contact handling (PHP) and helpers

## Customize

- Edit `index.html` to change content (name, bio, skills, projects). Replace text and update `data-` attributes used by `Typed.js`.
- Replace `assets/img/my-profile-img.jpg` for the profile avatar, and other images inside `assets/img/portfolio/` for project images.
- Adjust styles in `assets/css/main.css` or edit vendor CSS in `assets/vendor/`.
- Add or remove sections by editing the HTML directly. For extra scripts, update `assets/js/main.js`.
